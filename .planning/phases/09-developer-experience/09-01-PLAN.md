<task type="auto">
  <name>Task 3: Update docker-compose.yml for MEM_LIMIT variable</name>
  <files>docker-compose.yml</files>
  <action>
Modify docker-compose.yml to use MEM_LIMIT environment variable with default.

Change the app service mem_limit from:
  `mem_limit: 2g`
To:
  `mem_limit: ${MEM_LIMIT:-2g}`

This allows users to override via .env file while keeping 2g as sensible default.

Also add .env file reference comment at top of file:
```yaml
# Configuration: Copy .env.example to .env to customize settings
# See .env.example for all available options
```
  </action>

Run `docker compose config` and verify MEM_LIMIT interpolation works.

  </verify>
