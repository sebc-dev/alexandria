<task type="auto">
  <name>Task 2: Create HTTP/SSE transport profile</name>
  <files>src/main/resources/application-http.yml</files>
  <action>
Create application-http.yml with HTTP/SSE MCP configuration:

```yaml
# HTTP/SSE MCP Server profile - for containerized deployment
spring:
  main:
    # Enable web server for HTTP/SSE transport
    web-application-type: servlet
    banner-mode: console

  ai:
    mcp:
      server:
        name: alexandria
        version: 0.1.0
        type: SYNC
        # SSE transport for remote clients
        stdio: false

server:
  port: 8080
  shutdown: graceful

spring:
  lifecycle:
    timeout-per-shutdown-phase: 30s

management:
  endpoints:
    web:
      exposure:
        include: health
  endpoint:
    health:
      show-details: never
      probes:
        enabled: true

logging:
  level:
    fr.kalifazzia.alexandria: INFO
    org.springframework: WARN
```

Key configuration:
- `web-application-type: servlet` enables embedded Tomcat
- `stdio: false` switches to HTTP/SSE (webmvc starter auto-configures /sse endpoint)
- `server.shutdown: graceful` handles SIGTERM properly
- Actuator health endpoint exposed for Docker health checks
  </action>
  <verify>File exists with correct YAML structure - `cat src/main/resources/application-http.yml`</verify>
  <done>application-http.yml created with SSE transport and graceful shutdown config</done>
</task>
