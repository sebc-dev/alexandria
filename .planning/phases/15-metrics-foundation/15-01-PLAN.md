  <name>Task 1: Add Prometheus registry and configure actuator</name>
  <files>pom.xml, src/main/resources/application.yml</files>
  <action>
1. Add micrometer-registry-prometheus dependency to pom.xml (no version - managed by Spring Boot BOM):
```xml
<dependency>
    <groupId>io.micrometer</groupId>
    <artifactId>micrometer-registry-prometheus</artifactId>
</dependency>
```

2. Add management configuration to application.yml (in default profile section, before the `---`):
```yaml
management:
  endpoints:
    web:
      exposure:
        include: health,prometheus
  metrics:
    distribution:
      percentiles-histogram:
        alexandria.search.duration: true
        alexandria.embedding.duration: true
```

This enables:
- /actuator/prometheus endpoint exposure
- Server-side percentile histograms for our custom timers (p50/p95/p99 calculated by Prometheus)
  </action>

- `grep -A5 "management:" src/main/resources/application.yml` shows config
  </verify>
  <done>
- pom.xml contains micrometer-registry-prometheus dependency
- application.yml exposes prometheus endpoint with histogram config
  </done>

  <action>
1. Update SearchService constructor to inject MeterRegistry and create Timer:
   - Add `import io.micrometer.core.instrument.MeterRegistry;`
   - Add `import io.micrometer.core.instrument.Timer;`
   - Add field: `private final Timer searchTimer;`
   - Add MeterRegistry parameter to constructor
   - Create timer in constructor:
     ```java
     this.searchTimer = Timer.builder("alexandria.search.duration")
         .description("Time spent executing search operations")
         .publishPercentileHistogram()
         .register(meterRegistry);
     ```

2. Wrap search logic in timer.record():
   - In `search(String query, SearchFilters filters)` method, wrap the embedding+search calls:
     ```java
     return searchTimer.record(() -> {
         float[] queryEmbedding = embeddingGenerator.embed(query);
         List<SearchResult> results = searchRepository.searchSimilar(queryEmbedding, filters);
         log.info("Search for '{}' returned {} results", truncate(query, 50), results.size());
         return results;
     });
     ```
   - In `hybridSearch(String query, HybridSearchFilters filters)` method, wrap similarly:
     ```java
     return searchTimer.record(() -> {
         float[] queryEmbedding = embeddingGenerator.embed(query);
         List<SearchResult> results = searchRepository.hybridSearch(queryEmbedding, query, filters);
         log.info("Hybrid search for '{}' returned {} results", truncate(query, 50), results.size());
         return results;
     });
     ```

3. Update SearchServiceTest to inject SimpleMeterRegistry:
   - Add `import io.micrometer.core.instrument.simple.SimpleMeterRegistry;`
   - Add field: `private MeterRegistry meterRegistry;`
   - In setUp(), initialize: `meterRegistry = new SimpleMeterRegistry();`
   - Pass meterRegistry to SearchService constructor

4. Add test to verify timer recording:
   ```java
   @Test
   void search_recordsTimerMetric() {

       when(searchRepository.searchSimilar(any(), any())).thenReturn(List.of());

       searchService.search("test", 5);

       Timer timer = meterRegistry.get("alexandria.search.duration").timer();
       assertThat(timer.count()).isEqualTo(1);
   }
   ```
