**EvaluationService.java** - Orchestration service in core/evaluation:
- `@Service` annotation
- Constructor injection: `SearchService`, `GoldenDatasetLoader`
- Configurable pass thresholds (default: Precision@5 >= 0.5, NDCG@10 >= 0.5)

**evaluate(Path datasetPath) method:**
1. Load golden dataset: `datasetLoader.load(datasetPath)`
2. For each GoldenQuery:
   a. Execute search: `searchService.hybridSearch(query, 20)` (k=20 for Recall@20)
   b. Extract unique document IDs from SearchResult list:
      ```java
      List<UUID> retrievedDocIds = results.stream()
          .map(SearchResult::documentId)
          .distinct()  // Multiple chunks from same doc
          .toList();
      ```
   c. Create Set&lt;UUID&gt; relevant from golden.expectedDocIds()
   d. Calculate all metrics using RetrievalMetrics:
      - precisionAtK(retrieved, relevant, 5)
      - precisionAtK(retrieved, relevant, 10)
      - recallAtK(retrieved, relevant, 10)
      - recallAtK(retrieved, relevant, 20)
      - reciprocalRank(retrieved, relevant)
      - ndcgAtK(retrieved, relevant, 10)
   e. Create QueryEvaluation record
3. Aggregate results:
   - Overall: average all metrics across all queries
   - By question type: group by questionType, average within group
   - Determine passed based on thresholds
4. Return EvaluationReport with evaluatedAt = Instant.now()

**IMPORTANT:** Extract documentId from SearchResult, NOT childChunkId. The golden dataset contains document IDs, not chunk IDs.
