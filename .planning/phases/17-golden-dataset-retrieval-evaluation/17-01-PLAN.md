- Constructor injection of `ObjectMapper` (already configured in JacksonConfig)
- `@Component` annotation for Spring detection
- `load()` method:
  1. Use `Files.newBufferedReader(datasetPath)` for proper encoding
  2. Read line by line with `readLine()`
  3. Track line numbers for error messages
  4. Skip blank lines (`line.isBlank()`)
  5. Parse each line with `objectMapper.readValue(line, GoldenQuery.class)`
  6. Catch `JsonProcessingException` and wrap with line number context: `"Invalid JSON at line N: message"`
  7. Return immutable list of GoldenQuery objects

Do NOT load entire file into memory - use streaming line-by-line approach.
